# PQ-XDH-Secure-Messaging-Conference-Keying
A secure messaging client implementing the PQXDH protocol by combining Ed25519 and Kyber1024 for quantum-resistant encryption. It utilizes a SHA3-256 KDF chain for forward secrecy and HMAC-SHA256 for integrity. I also engineered a Conference Keying protocol for group chat and automated workflows like Kyber decapsulation and AES-CTR encryption.
